# functional-design-in-phel-lang

関数型デザインをphel-langで写経します。



## Development

### Open shell

```bash
docker compose build
docker compose run --rm php_cli bash
```

### install dependencies

```bash
composer install
```
### run phel

```bash
vendor/bin/phel run src/3-1.phel
```

### execute tests

```bash
composer test
```
